# Connect Amravati - District Communication Portal

A modern multilingual login portal for Collector Office, Amravati built with Vite + React.

## 🚀 Features

- 🌐 **Multi-language Support** (English, मराठी, हिन्दी)
- 🎨 **Beautiful UI** with Tailwind CSS
- ✨ **Smooth Animations** using Framer Motion
- 🔐 **Form Validation** with toast notifications
- 📱 **Fully Responsive** design
- 🏛️ **Government-style** professional theme
- ⚡ **Lightning Fast** with Vite

## 🛠️ Tech Stack

- React 18
- Vite 4
- Tailwind CSS 3
- Framer Motion 10
- React Icons 4
